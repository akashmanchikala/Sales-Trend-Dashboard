# 📊 Sales Trend Dashboard

## 🔍 Problem Statement
Retail sales data is often fragmented across multiple sources, making it difficult to track KPIs, identify top-performing products, and analyze regional performance.

## 🚀 Solution
Developed a business intelligence dashboard to consolidate and analyze sales data, enabling stakeholders to make data-driven decisions.

---

## 🛠️ Tech Stack
- Python (Pandas, NumPy)
- SQL
- Power BI
- Tableau
- Excel

---

## 📂 Project Workflow
1. Data Collection (CSV / Excel files)
2. Data Cleaning using Python
3. Data Aggregation & Querying using SQL
4. Dashboard Creation in Power BI & Tableau
5. Insights & KPI Tracking

## 📊 Key Features
- Sales performance tracking
- Region-wise analysis
- Top-selling products identification
- Interactive dashboards
